# Bike Buyers Data Analysis

## Overview

This project analyzes customer data to identify factors that influence bike purchasing decisions.

The analysis includes data cleaning, exploration, visualization, and basic machine learning techniques to uncover patterns in customer behavior.

---

## Features

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Data visualization using graphs
* Statistical insights from customer data
* Basic machine learning models (sklearn, xgboost)

---

## Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* XGBoost
* Jupyter Notebook (Google Colab)

---

## Project Structure

* `ONO_Mid_term_exercise_2023.ipynb` – Main analysis notebook

---

## How to Run

### Option 1 – Google Colab

1. Open the notebook in Google Colab
2. Upload the dataset (CSV file if needed)
3. Run all cells

### Option 2 – Local Machine

1. Install Python
2. Install dependencies:

   ```
   pip install pandas numpy matplotlib scikit-learn xgboost notebook
   ```
3. Run:

   ```
   jupyter notebook
   ```

---

## Key Insights

* Income has a strong impact on bike purchasing decisions
* Age and demographic factors influence customer behavior
* Data visualization helps identify trends and patterns
